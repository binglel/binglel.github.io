快速搭建与使用SOCKS5代理服务

November 14, 2020 185

启动

git clone https://github.com/binglel/socks5_python
cd socks5_python
python socks5.py -h
python socks5.py start --port=1080 --auth=bulabula:31415926

停止

python socks5.py stop
